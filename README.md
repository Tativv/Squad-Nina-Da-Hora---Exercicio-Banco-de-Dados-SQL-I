# 📊 Análise de Dados com SQL - Base Universitária

Projeto desenvolvido no Bootcamp de Business Intelligence (Instituto Localiza) com foco em consultas SQL e análise de dados.

---

## 📌 Visão geral

Este projeto utiliza uma base de dados acadêmica para explorar e analisar informações relacionadas a alunos, cursos e matrículas.

Foram aplicadas consultas SQL para extração de insights, organização de dados e análise de desempenho.

## 📂 Base de dados

O projeto utiliza um banco SQLite contendo:

- **ALUNOS**: dados dos estudantes  
- **CURSOS**: informações dos cursos  
- **MATRICULAS**: relação entre alunos e cursos  

---

## 📊 Análises realizadas

- Consultas com filtros e ordenação  
- Transformação e classificação de dados  
- Agregações (média, contagem, agrupamentos)  
- JOINs entre múltiplas tabelas  
- Identificação de padrões e desempenho acadêmico  

---

## 🔍 Destaques

- Identificação do aluno com melhor desempenho  
- Análise de média de notas por grupo  
- Avaliação da distribuição de cursos por departamento  

---

## ▶️ Como executar

1. Baixe o arquivo `DB_Universidade.db`  
2. Abra em um cliente SQLite (DBeaver, SQLiteStudio, etc.)  
3. Execute o script `respostas.sql`
